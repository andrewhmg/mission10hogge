# Mission #10 – Bowling League (Marlins & Sharks)

**Date:** March 13, 2025

This project is Mission #10 for IS 413. It shows bowler information from the Bowling League database for the **Marlins** and **Sharks** teams only. The site is built with an ASP.NET Core API and a React front-end that displays each bowler’s name, team, address, city, state, zip, and phone number.
